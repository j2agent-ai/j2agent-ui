# 样式约定

样式入口：`src/styles/markdown.scss`（在 `main.ts` 全局引入）。聊天气泡根节点 class：**`.message-md`**。

## 图表容器

| 变量 / 类 | 说明 |
|-----------|------|
| `--md-diagram-max-height` | 图表块最大高度，默认 `min(360px, 42vh)` |
| `.md-diagram` / `.md-diagram-body` | 毛玻璃底、圆角、居中 |
| `.md-diagram-body--wide` | 宽于气泡时横向滚动（thin scrollbar） |
| `.md-diagram-xychart` | Mermaid xychart 块标记（便于定向覆盖） |
| `.md-diagram-fit` | SVG 外包层，配合 JS 设宽高 |
| `.md-diagram-error` | 危险色背景 + details 错误区 |

JS 中 `MARKDOWN_DIAGRAM_MAX_HEIGHT_FALLBACK = 360` 须与 SCSS 回退一致。

## 表格

- 外包 `.md-table-wrap`，横向滚动
- `.md-table` / `.md-thead` / `.md-tbody` 使用玻璃态表头与斑马纹

## 代码

| 场景 | 选择器 | 样式要点 |
|------|--------|----------|
| 行内 code | `code:not(pre code)` | 浅色标签底 `--n-inline-code-*`，无 `backdrop-filter` |
| 围栏代码 | `pre` / `pre code` | 深色玻璃面板，可横向滚动 |

行内 code 勿复用围栏/思考块的深色毛玻璃，否则在浅色气泡上会出现模糊脏边。

## 链接与标题

- 链接继承气泡文字色，hover 主色
- `h1`–`h6` 使用 `--n-font-size-*` 阶梯

## ChatView 局部覆盖

`ChatView.vue` 的 `:deep(.message-md …)` 仅做颜色继承与图片 hover；**不要**再把行内 `code` 强制设为深色字，以免与 `markdown.scss` 冲突。

## 主题色

图表区背景 `--n-color-diagram-bg` / 边框 `--n-color-diagram-border` 在 `var.scss` 中定义，与全局微蓝毛玻璃 tint 分离，避免报表图被染色面板影响对比度。

分类色板 `--n-color-diagram-palette-1` … `8` 与 `diagramTheme.ts` 中 `DIAGRAM_COLOR_PALETTE` 保持一致；Mermaid / Vega-Lite / PlantUML 的 JS 主题亦出自该模块。
